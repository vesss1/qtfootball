# 進球練習記錄功能使用指南 (Goal Practice Tracker User Guide)

## 功能概述 (Feature Overview)

這個功能允許教練或球員記錄從特定距離踢球的練習數據，包括：
- 嘗試次數
- 成功進球數
- 花費時間
- 成功率分析

This feature allows coaches or players to record practice data for kicking from specific distances, including:
- Number of attempts
- Successful goals
- Time spent
- Success rate analysis

## 使用步驟 (Usage Steps)

### 1. 新增球員 (Add Player)
首先在頂部表單中輸入球員資訊：
- 球員姓名 (Player Name)
- 位置 (Position): 前鋒/中場/後衛/守門員
- 年齡 (Age): 15-50歲

點擊「新增球員」按鈕。

### 2. 開始練習記錄 (Start Practice Recording)

在「進球練習記錄」區域：
1. 從下拉選單選擇球員
2. 設定距離（碼）- 預設為 8 碼
3. 設定計劃嘗試次數 - 預設為 5 次
4. 點擊「開始練習」按鈕

### 3. 記錄每次嘗試 (Record Each Attempt)

練習開始後：
- 點擊「記錄成功進球」- 當球進入球門時
- 點擊「記錄未進球」- 當球未進入球門時

系統會即時顯示：
- 當前進度（已嘗試/計劃嘗試）
- 成功次數
- 當前成功率
- 經過時間

### 4. 結束練習 (End Practice)

- 點擊「結束練習」按鈕
- 系統會顯示本次練習的完整摘要
- 數據會自動保存到球員的練習歷史中

### 5. 查看數據分析 (View Data Analysis)

在「數據分析」區域：
- 點擊「顯示統計數據」按鈕
- 表格會顯示所有練習記錄，包括：
  - 球員姓名
  - 距離（碼）
  - 總嘗試次數
  - 成功進球數
  - 成功率（%）
  - 花費時間（秒）

## 範例場景 (Example Scenario)

### 場景：從 8 碼位置踢 5 球 (Kick 5 balls from 8 yards)

1. **準備階段**
   - 新增球員「張三」，位置「前鋒」，年齡 25
   - 選擇球員「張三」
   - 設定距離：8 碼
   - 設定嘗試次數：5 次
   - 點擊「開始練習」

2. **練習過程**
   - 第1球：進球 → 點擊「記錄成功進球」（成功率：100%）
   - 第2球：未進 → 點擊「記錄未進球」（成功率：50%）
   - 第3球：進球 → 點擊「記錄成功進球」（成功率：66.7%）
   - 第4球：進球 → 點擊「記錄成功進球」（成功率：75%）
   - 第5球：未進 → 點擊「記錄未進球」（成功率：60%）

3. **結果顯示**
   ```
   練習完成！
   
   球員: 張三
   距離: 8 碼
   總嘗試: 5 次
   成功進球: 3 次
   成功率: 60.0%
   花費時間: 45.3 秒
   ```

## 技術細節 (Technical Details)

### 資料結構 (Data Structures)

```cpp
struct PracticeSession {
    int distance;           // 距離（碼）
    int totalAttempts;      // 總嘗試次數
    int successfulGoals;    // 成功進球數
    qint64 timeSpent;       // 花費時間（毫秒）
    QString playerName;     // 球員姓名
};

struct Player {
    QString name;
    QString position;
    int age;
    QVector<PracticeSession> practiceSessions;  // 練習歷史
};
```

### 計時機制 (Timing Mechanism)
- 使用 `QElapsedTimer` 進行高精度計時
- 開始練習時啟動計時器
- 結束練習時記錄總耗時（毫秒）
- 統計表格中顯示為秒（保留一位小數）

### 即時更新 (Real-time Updates)
- 每次記錄嘗試時立即更新顯示
- 動態計算成功率
- 顯示經過時間
- 到達計劃次數時自動提示

## 建置與執行 (Build and Run)

### 前置需求 (Prerequisites)
- Qt 5.x 或 Qt 6.x
- C++17 編譯器
- qmake 或 CMake

### 建置步驟 (Build Steps)
```bash
cd foot
qmake foot.pro
make
./foot  # Linux/Mac
# or
foot.exe  # Windows
```
